# Com(Plex) (Sim)ulation toolbox

Fast and general toolbox for simulation of complex adaptive systems written in cython.
_N.b._ this is a work in progress

## Dependencies

- python >= 3.8
- cython >=.28
- numpy >= 1.18.1
... see req.txt
